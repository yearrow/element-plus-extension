### 表格分类
- 不分页
预期行数较少（<=200）无需分页时使用
- 后分页
预期行数一般（>200 且  <= 800）时使用
- 先分页
预期行数过大（>800 且  <= 10000）时使用
- 翻页表格
预期行数巨大（>10000）使用
### 列头
- 列头统一居中，列头标题尽量不换行
- 根据内容设置列宽，尽量显示完整，留一列较长列作为自适应列，表格铺满；如果实在列太少，可以不用铺满；如果列太多，超过屏幕长度，则不用使用自适应列，每列给宽度即可
### 表格内容
1. 排列方式
- 1个tag的情况下居中，多个tag则靠左
- 序号、选择器、操作按钮 居中
- 表格内容文字靠左，数字靠右
2. 按钮
- 表格行内按钮大小都使用small
3. 规范
- 表格的行高固定，如果内容的字数过长，显示气泡提示框
4. 排序
