### api

#### 属性

| 属性        | 说明       | 类型 | 可选值 | 默认值 |
| :--------- |:--------| :-----| :-----| :-----|
| show-header	| 显示头部	| boolean	| 	| true |
| title	| 标题	| string	| 	|   |
| border| 显示边框	| boolean	|	|  false|
| shadow| 显示阴影	| boolean	|	|  false |
| paddingSize	| 内边距大小	| string	| small/base/large |  |
| clearPadding	| 清除内边距	| string[]	| left/right/top/bottom	| false |
